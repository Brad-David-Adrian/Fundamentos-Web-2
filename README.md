# Fundamentos-Web-2

## Descripción del Proyecto

Este proyecto es un portafolio personal desarrollado como parte del curso de Fundamentos Web 2. El sitio web presenta información sobre el estudiante Adrian David Brad Rus, incluyendo su biografía, habilidades, proyectos destacados y datos de contacto. El diseño utiliza tecnologías modernas para crear una interfaz atractiva y funcional.

## Tecnologías Utilizadas

- **HTML5**: Estructura del contenido web.
- **CSS3**: Estilos y diseño visual, incluyendo un tema "cosmic" personalizado.
- **Bootstrap 5**: Framework CSS para diseño responsivo y componentes UI.
- **JavaScript**: Interactividad básica y funcionalidades dinámicas.
- **Three.js**: Para elementos 3D interactivos (usado en proyectos destacados).

## Estructura del Proyecto

- `index.html`: Página principal con navegación orbital 3D.
- `Html/`: Carpeta con páginas HTML adicionales (about.html, contacts.html, etc.).
- `Css/Styles.css`: Hoja de estilos personalizada.
- `Js/Code.js`: Scripts JavaScript para funcionalidades interactivas.
- `images/`: Carpeta para imágenes del portafolio (fotos personales, etc.).

## Instrucciones de Uso

1. Clona el repositorio: `git clone [URL del repositorio]`
2. Abre `index.html` en un navegador web moderno.
3. Navega por las secciones usando el menú de navegación.
4. Para la galería personal en `about.html`, agrega tus fotos en la carpeta `images/` y actualiza las rutas en el código HTML.

## Características Principales

- Diseño responsivo adaptable a diferentes dispositivos.
- Tema visual "cosmic" con efectos de texto y fondos.
- Galería de fotos personal en la página "Acerca de".
- Enlaces a proyectos destacados y redes sociales.
- Formulario de contacto funcional.
